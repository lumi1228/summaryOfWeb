#### 1. 什么是字典

* 与集合类似，字典也是一种存储唯一值的数据结构，但他以键值对的形式存储
* ES6中有字典的数据结构，为Map
* 字典常用操作：键值对的增删改查
```
const m=new Map();
// 增
m.set('a','aa');
m.set('b','bb');
//删
m.delete('b')
// m.clear(); //删除所有数据
//改
m.set('b', 'bbb');
// 查
m.get('a')
```
